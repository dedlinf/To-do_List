Weekly Calendar TodoList
Это веб-приложение для планирования задач на неделю. Проект разработан в рамках ознакомительной практики на кафедре ПОВТ.

О проекте
Приложение представляет собой интерактивный календарь-планировщик.

Бэкенд: Go (Golang) — обеспечивает работу сервера и сохранение данных в файл tasks.json.

Фронтенд: HTML5, CSS3, JavaScript (Vanilla) — динамический интерфейс для отображения задач и взаимодействия с сервером.

Как запустить проект
Установите Go (если еще не установлен) с официального сайта go.dev.

Скачайте репозиторий:

Bash 
cd To-do_List
Запустите сервер:
В терминале, находясь в папке проекта, выполните:

Bash
go run main.go
Откройте приложение:
Перейдите в браузере по адресу: http://localhost:8080

Структура проекта
main.go — основной файл сервера на языке Go.

tasks.json — локальная база данных задач в формате JSON.

/static — папка с файлами интерфейса (HTML, CSS, JS).

Технологический стек
Язык: Go

Верстка: HTML5, CSS3

Скрипты: JavaScript (Fetch API)
